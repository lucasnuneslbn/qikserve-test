# qikserve test

## Pré-requisitos

Nodejs e npm instalados na máquina

### Rodando a aplicação

```
npm start
```

Agora abra o browser e digite: [`localhost:8000/index.html`][local-app-url].

* O teste em si está na aba `PDV`


## Testando a aplicação

```
npm test
```
